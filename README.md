MyBoba is an iOS boba-ordering application built with Swift, UIKit, and Storyboard. It simulates a boba tea shop experience, allowing users to customize drinks, view real-time prices, and manage a cart through a clean multi-screen interface.

## Features
- 10+ customizable boba drink options
- Real-time price calculation based on selections
- Multi-screen navigation using segues
- MVC architecture for clean code organization
- Functional cart system for managing orders
- Responsive UIKit-based user interface

## Tech Stack
- Swift
- UIKit (UITableView, UIStackView, Segues, Storyboard)  
- Storyboard
- Xcode
- MVC Design Pattern

## Purpose
This project was built to strengthen iOS development skills, including UI design, application architecture, and state management. It provides a user-friendly experience that allows users to customize and order boba drinks in advance, simulating a real-world ordering system while following best practices in Swift and UIKit.
